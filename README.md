# Retaurant Reviews App - Stage 1
---

This is the first one of the three stage **Restaurant Reviews** project. The main goal of this stage is converting a static webpage to a mobile-ready web application.


### Installation

1. Clone the project from [GitHub repo](https://github.com/udacity/mws-restaurant-stage-1)
2. Run server with "python3 -m http.server"
3. With your server running, visit the site: `http://localhost:8000`
